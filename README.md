## NexusIQ AI — Multi-Agent Business Intelligence Platform

> *Ask a question in plain English. Get validated insights from SQL, documents, and live web data — in seconds.*

An enterprise-grade **multi-agent AI platform** that routes natural language business questions through a coordinated network of specialized AI agents — delivering SQL analysis, document intelligence, and real-time web data in one unified, cross-validated response.

### How It Works

```
User Question → Fusion Agent (LLM Router)
                     │
          ┌──────────┼──────────┐
          ▼          ▼          ▼
      SQL Agent   RAG Agent  Web Agent
    (100K rows)  (23 PDFs)  (Live prices)
          │          │          │
          └──────────┴──────────┘
                     │
              Fused Answer + Confidence Badges
```

### What Makes It Different

| Capability | Details |
|---|---|
| **SQL Agent** | Natural language → SQL → Interactive Plotly charts |
| **RAG Agent** | Hybrid BM25 + vector search across 23 internal PDF documents |
| **Web Agent** | Live competitor pricing via Selenium + BeautifulSoup |
| **Fusion Agent** | Orchestrates all agents, cross-validates facts, shows confidence scores |
